Here will describe in detail how users can customize the video display interface.
First, explain how to display the video list in the browser in the program.

  1. The program will load the workspace/config/site/index.html file in the directory where the program is located into the browser.
  2. Before loading the index.html file, it will automatically inject a series of functions into the browser. These functions will exist in the browser as a JS-Java bridge, that is, you can call Java services through JS scripts (specifically Services are given in the form of a table below).
  3. When the Html is loaded, it will use JavaScript to call the Java service to dynamically generate Html code to display the video information.

So anyone who is not satisfied with the video list interface displayed by the current browser, or expects more functions, can modify workspace/config/site/index.html in the directory where the program is located, and then in the browser interface Click the "Reload Browser" menu to view the effect (you need to make sure that the browser is not using the browser's own menu, which can be modified in the preferences).

Name and parameters
Function description
Example
int queryMovieCountFunction()
Query the number of videos under the current site (if there is a filter condition, it will be the filtered result)
JSON queryMovieListFunction(int currentPage,int pageCount)
Query the video content in the specified page under the current site, and the result will be returned in JSON format, please refer to the following content
queryMovieListFunction(10,40) If each page contains 40 records, what data is on page 10
void downloadMovieListFunction(String movieIdList)
Download the video with the specified ID number, the parameter is a string, which is a combination of ID numbers, separated by "," or "|"
queryMovieListFunction("11456,23768") downloads two videos with ID numbers 11456 and 23768
void selectMovieFunction(int movieId)
is equivalent to a message event, notifying the program that the user selects a video, and the information of this video can be displayed in the attribute view.
selectMovieFunction(12345) The video with the notification program ID number 12345 is selected
