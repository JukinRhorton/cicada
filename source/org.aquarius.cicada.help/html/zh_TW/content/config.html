配置說明

目前軟件提供了四個首選內容的配置，下面介紹一下幾個重要的配置項內容。

下載說明：

  1. 指定ffmpeg命令，對于使用HLS協議的流媒體來說，下載完成后，需要使用ffmpeg進行合并，可以在這里配置好相應的命令行。如果已經把ffmpeg加到當前環境里，使用默認配置即可。否則需要指定ffmpeg執行文件的路徑，如"C:\MyTools\ffmpeg\bin\ffmpeg.exe -f concat -safe 0 -i "{0}" -c copy "{1}"，配置時，請一定要正確設置"{0}"和"{1}"，前者是對應的HLS列表文件，后者是輸出的媒體文件。這兩個參數均由系統自動填寫。
  2. 配置調用系統命令，并將"合并HLS完成后刪除源文件"勾選后，即可自動清除合并后多余的HLS分片文件。
  3. 很多站點對線程連接數均有約束，所以如果出現因為線程連接數過多而出錯的問題，可以在最下面的表格里為指定的站點設置連接數限制。假如www.google.com的連接數為10，設置時只需要用google或者google.com作為站點關鍵字即可，因為系統會使用字符串包含的方式進行判斷。

界面說明：

  1. 如果感覺系統資源占用過高，或者系統出現不穩定的情況，可以將"使用Chromium內核"這一配置取消，重啟后即可禁用Chrominum，可以提高系統穩定性。
  2. 不管是Mp4還是HLS流媒體，有很多播放器都是可以直接播放的，如果想直接使用本機的瀏覽器查看視頻，那么可以通過設置"啟動播放器命令"來快捷啟用該功能。默認使用了PotPlayer，你可以隨意指定自己偏愛的播放器，只需要注意將參數"{0}"放在正確的位置即可，因為系統會自動將遠程播放的URL地址填寫在這里后再執行。

 

對于"視頻"和"站點"管理比較簡單，不再詳細說明。
